# Agent & LLM Skill

> Skill for designing and managing AI Agent logic and LLM interactions.

## Core Principles
1. Safety First: Use budget guards and output validators.
2. Deterministic Logic: Keep the agent flow predictable.
3. HITL (Human-In-The-Loop): Request user feedback for low-confidence actions.

## Workflow
1. Prompt Design: Create and test prompts in a sandbox.
2. Tool Integration: Define clear tool schemas and error handling.
3. Execution: Run the agent with monitoring and logging.
4. Optimization: Refine prompts based on feedback.
